# TheUrbanReview
📰Esse projeto é sobre um site jornalistico, criado para repassar informações virtualmente para o público.

---

## 📸Captura de Tela
![Prévia do Projeto](src="/src/img/CapturaDeTela.png)

---

## 🔧Funcionalidades
* Botão de modo escuro
* Aba de comentarios
* Selecionamento de paginas sobre as noticias
* Mundança de cor para a pagina de esportes

---

## 🖥️Tecnologias ultilizadas
- HTML
- CSS
- JavaScript
- GIT e Github
- VS code

---

## ❓Como rodar?
* 1-Baixe o repositório (Clique no botão "Code" Download Zip)
* 2-Extraia a pasta
* 3-Abra o arquivo index.html no navegador
